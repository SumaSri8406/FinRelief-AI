# Documentation Directory

Welcome to the documentation folder for **FinRelief AI**. 

This directory is reserved for system designs, API schemas, deployment manuals, and other technical documentations.

## Directory Layout
- `api_spec.md`: OpenAPI / Swagger schemas and endpoints overview.
- `database_design.md`: Detailed database entity relations, constraints, and optimization specifications.
- `deployment_guide.md`: Steps for deploying the FastAPI backend and React frontend to production environments (such as AWS, Heroku, or Vercel).
